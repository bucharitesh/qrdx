# Introduction

QRDX is a React library for creating customizable QR codes. Add it to your project, customize colors and patterns, and download in multiple formats.

## Key Features

Here are some of the features QRDX offers:

- [Custom colors and gradients](/docs/features/colors-gradients)
- [Built-in templates](/docs/features/templates) for instant styling
- [Logo integration](/docs/features/logos) with automatic positioning
- [Multiple patterns](/docs/features/patterns) for unique designs
- [Download as PNG, JPG, SVG, or PDF](/docs/features/downloads)
- [QR code detection](/docs/qr-detection) from images
- Full TypeScript support with type safety
- Server-side rendering compatible

## Quick Example

```tsx
import { QRCode } from 'qrdx';

export default function App() {
  return (
    <QRCode
      value="https://example.com"
      size={512}
      fgColor="#000000"
      bgColor="#ffffff"
    />
  );
}
```
