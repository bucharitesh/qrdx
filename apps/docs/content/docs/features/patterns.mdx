# Patterns

Change the shape of QR code elements for unique designs.

## Body Patterns

Control the shape of data dots:

```tsx
import { QRCode } from 'qrdx';

<QRCode value="https://example.com" bodyPattern="circle" size={256} />
<QRCode value="https://example.com" bodyPattern="square" size={256} />
<QRCode value="https://example.com" bodyPattern="diamond" size={256} />
<QRCode value="https://example.com" bodyPattern="rounded" size={256} />
<QRCode value="https://example.com" bodyPattern="pacman" size={256} />
<QRCode value="https://example.com" bodyPattern="circle-mixed" size={256} />
<QRCode value="https://example.com" bodyPattern="clean-square" size={256} />
```

## Corner Eye Patterns

Control the shape of corner markers:

```tsx
<QRCode value="https://example.com" cornerEyePattern="square" />
<QRCode value="https://example.com" cornerEyePattern="rounded" />
<QRCode value="https://example.com" cornerEyePattern="circle" />
```

## Corner Eye Dot Patterns

Control individual dots in corner markers:

```tsx
<QRCode value="https://example.com" cornerEyeDotPattern="square" />
<QRCode value="https://example.com" cornerEyeDotPattern="rounded" />
<QRCode value="https://example.com" cornerEyeDotPattern="circle" />
```

## Combine Patterns

Mix different patterns for unique designs:

```tsx
<QRCode
  value="https://example.com"
  bodyPattern="rounded"
  cornerEyePattern="circle"
  cornerEyeDotPattern="circle"
  size={256}
/>
```

## Performance

### Fast Patterns
- `circle` — Excellent performance
- `square` — Very fast
- `clean-square` — Fast

### Medium Patterns
- `rounded` — Good balance
- `diamond` — Moderate

### Complex Patterns
- `pacman` — Higher processing
- `circle-mixed` — Variable complexity

## Pattern Selection

### For Brand Identity
```tsx
// Minimal, professional
<QRCode
  value="https://example.com"
  bodyPattern="rounded"
  cornerEyePattern="square"
/>

// Modern, tech-focused
<QRCode
  value="https://example.com"
  bodyPattern="circle"
  cornerEyePattern="rounded"
/>
```

### For Marketing
```tsx
// Eye-catching
<QRCode
  value="https://example.com"
  bodyPattern="pacman"
  cornerEyePattern="circle"
/>

// Elegant
<QRCode
  value="https://example.com"
  bodyPattern="diamond"
  cornerEyePattern="rounded"
/>
```

### For Print
```tsx
// High contrast
<QRCode
  value="https://example.com"
  bodyPattern="square"
  cornerEyePattern="square"
/>

// Smooth, professional
<QRCode
  value="https://example.com"
  bodyPattern="rounded"
  cornerEyePattern="rounded"
/>
```
