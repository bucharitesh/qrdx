# Templates

Apply professional QR code styles instantly.

## What Are Templates?

Templates combine colors, patterns, and visual styles into ready-to-use packages.

## Available Templates

### Default
Classic black and white.

```tsx
import { QRCode } from 'qrdx';

<QRCode value="https://example.com" templateId="default" size={256} />
```

### Arrow
Modern geometric design.

```tsx
<QRCode value="https://example.com" templateId="Arrow" size={256} />
```

### Standard Box
Clean, structured design.

```tsx
<QRCode value="https://example.com" templateId="StandardBox" size={256} />
```

### Square Border
Bordered design with defined edges.

```tsx
<QRCode value="https://example.com" templateId="SquareBorder" size={256} />
```

### Striked Box
Dynamic design with diagonal elements.

```tsx
<QRCode value="https://example.com" templateId="StrikedBox" size={256} />
```

### Halloween
Seasonal Halloween theme.

```tsx
<QRCode value="https://example.com" templateId="Halloween" size={256} />
```

## Customize Templates

Override template defaults:

```tsx
<QRCode
  value="https://example.com"
  templateId="Arrow"
  fgColor="#your-brand-color"
  hideLogo={false}
  logo="/your-logo.png"
  size={512}
/>
```

## Template Registry

### Get All Templates

```tsx
import { getAllTemplates } from 'qrdx';

const templates = getAllTemplates();
```

### Get Template IDs

```tsx
import { getTemplateIds } from 'qrdx';

const ids = getTemplateIds();
// Returns: ['default', 'Arrow', 'StandardBox', ...]
```

## Custom Templates

Create your own templates:

```tsx
const brandTemplate = {
  id: 'brand-template',
  name: 'Brand Template',
  colors: {
    fgColor: '#1a56db',
    bgColor: '#ffffff',
    eyeColor: '#0d47a1',
    dotColor: '#3b82f6'
  },
  bodyPattern: 'rounded',
  cornerEyePattern: 'rounded',
  cornerEyeDotPattern: 'rounded'
};

<QRCode
  value="https://example.com"
  customTemplate={brandTemplate}
  size={256}
/>
```

## When to Use Templates

- **Brand Consistency** — Ensure all QR codes match
- **Quick Setup** — Get professional results fast
- **Seasonal Campaigns** — Use themed templates
- **Batch Generation** — Apply consistent styles
