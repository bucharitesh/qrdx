# Logos

Add brand logos to QR codes while maintaining scannability.

## Basic Logo

```tsx
import { QRCode } from 'qrdx';

<QRCode
  value="https://example.com"
  size={512}
  hideLogo={false}
  logo="https://example.com/logo.png"
  level="Q"
/>
```

Always use higher error correction (`Q` or `H`) with logos.

## Advanced Control

```tsx
import { QRCodeCanvas } from 'qrdx';

<QRCodeCanvas
  value="https://example.com"
  size={512}
  imageSettings={{
    src: "https://example.com/logo.png",
    height: 80,
    width: 80,
    excavate: true,
    x: 216,
    y: 216,
    opacity: 0.9
  }}
  level="Q"
/>
```

## Size Guidelines

- **256px QR**: 32-48px logo
- **512px QR**: 64-96px logo
- **1024px QR**: 128-192px logo

Keep logos between 10-30% of QR code size.

## Logo Excavation

### What It Does

Excavation removes QR modules under the logo for better scanning.

```tsx
<QRCodeCanvas
  value="https://example.com"
  size={512}
  imageSettings={{
    src: "/logo.png",
    height: 80,
    width: 80,
    excavate: true  // Recommended for most logos
  }}
/>
```

### When to Use

- ✅ Logos with solid backgrounds
- ✅ Logos that might interfere with scanning
- ❌ Transparent logos (optional)
- ⚠️ Always test scanning on real devices

## Logo Formats

- **PNG** — Best for transparency
- **JPG** — Good for photos
- **SVG** — Scalable but may have loading issues
- **WebP** — Modern format, good compression

## Center Calculation

```tsx
const qrSize = 512;
const logoSize = 80;
const position = {
  x: (qrSize - logoSize) / 2,  // 216
  y: (qrSize - logoSize) / 2   // 216
};
```

## Best Practices

1. Use PNG format with transparency
2. Enable excavation for solid logos
3. Use `Q` or `H` error correction
4. Keep logo size reasonable (10-30%)
5. Test scanning on target devices
