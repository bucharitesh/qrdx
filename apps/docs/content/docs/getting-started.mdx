# Installation

Install QRDX and create your first QR code in minutes.

## Installation

```package-install
qrdx@latest
```

## Quick Start

Create your first QR code:

```tsx
import { QRCode } from 'qrdx';

export default function App() {
  return (
    <QRCode
      value="https://example.com"
      size={256}
    />
  );
}
```

That's it. You now have a working QR code.
